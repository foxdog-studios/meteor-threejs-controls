meteor-threejs-controls
=======================

Threejs Controls packaged for Meteor.

These are the controls that are provided in the [Three.js
examples](http://threejs.org/examples/)

Included are:

- EditorControls
- FirstPersonControls
- FlyControls
- OculusControls
- OrbitControls
- PathControls
- PointerLockControls
- TrackballControls
- TransformControls
